# Vasanthaprakash E - Data Analyst Portfolio

A secure, production-ready portfolio website built with TypeScript, Webpack, and modern web technologies.

## 🚀 Features

- **🛡️ Secure Architecture**: TypeScript compiled, minified, and obfuscated
- **📊 Dynamic Content**: Easy content management via JSON files
- **🎨 Modern UI**: Responsive design with smooth animations
- **💾 Data Collection**: Secure form submissions to Google Sheets
- **🔒 Data Encryption**: LocalStorage data is encrypted
- **📈 Analytics**: Built-in analytics tracking
- **🛠️ Admin Panel**: User-friendly content management interface
- **📱 Mobile Responsive**: Works on all devices

## 📋 Prerequisites

- Node.js (v16 or higher)
- npm (v7 or higher)
